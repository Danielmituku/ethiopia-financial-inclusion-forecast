# Notebooks

This directory contains Jupyter notebooks for the Ethiopia Financial Inclusion Forecasting project.

## Notebooks

### Task 1: Data Exploration and Enrichment
- `01_data_exploration.ipynb` - Initial exploration of the unified dataset

### Task 2: Exploratory Data Analysis
- `02_eda.ipynb` - Comprehensive EDA with visualizations and key insights

### Task 3: Event Impact Modeling
- `03_impact_modeling.ipynb` - Building event-indicator association matrix

### Task 4: Forecasting
- `04_forecasting.ipynb` - Access and Usage forecasts for 2025-2027

## Running Notebooks

```bash
# Install dependencies
pip install -r ../requirements.txt

# Start Jupyter
jupyter notebook
```

## Data Sources

All notebooks load data from:
- `../data/raw/` - Original starter dataset
- `../data/processed/` - Enriched and cleaned data
